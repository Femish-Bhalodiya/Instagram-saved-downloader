# Instagram-saved-downloader
Simple script to download instagram saved post to your system. 

# Getting Started

## 1.	Installation process
To get started with this project, you need to clone this repository to your local machine. Once you've cloned the repository, you can install the necessary dependencies by running the following command:

`npm install`

## 2.	Software dependencies
This project requires Node.js, axios, and some additional Node.js modules. The necessary Node.js modules are listed in the package.json file, which can be installed by running the npm install command.

## 3.	Latest releases
In development phase will update you soon.
